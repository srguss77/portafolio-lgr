# 🌐 Portafolio Personal — Luis Gustavo Ramírez Berganza

Portafolio personal desarrollado con HTML5, CSS3 y JavaScript vanilla.
Diseñado para mostrar habilidades y proyectos a clientes en Fiverr y otras plataformas freelance.

## 📁 Estructura del proyecto

```
portafolio-lgr/
├── index.html          ← Página principal
├── css/
│   └── styles.css      ← Todos los estilos
├── js/
│   └── main.js         ← Lógica e interacciones
└── README.md           ← Este archivo
```

## 🚀 Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/sr_gussrb/portafolio-lgr.git
   ```
2. Abre `index.html` en tu navegador.
3. No requiere instalación ni dependencias externas.

## ✨ Características

- Diseño 100% responsivo (móvil, tablet y escritorio)
- Menú hamburguesa en pantallas pequeñas
- Animaciones de entrada (fade up)
- Secciones: Hero, Sobre mí, Habilidades, Proyectos, Experiencia, Contacto
- Links a proyectos demo y repositorios de GitHub
